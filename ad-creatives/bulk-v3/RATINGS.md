# Ad Creative Ratings - Konteks Kourse Bulk v3

Rating scale: 1-10 on four dimensions
- **Scroll-stop:** Does it make you stop scrolling?
- **Emotional hit:** Does it trigger a feeling (fear, hope, curiosity, relief)?
- **Clarity:** Is the message instantly clear?
- **Design quality:** Typography, spacing, visual hierarchy

## Key Insights

### What Works
- **Big stats with specific insights** (87%, "the other 13%")
- **Visual differentiation** (split-tone backgrounds, inequality symbols)
- **Reframes + truth-bombs** ("built to filter you out", "never teach you")
- **Pattern interrupts** ("HONEST AD", two-tone split)
- **Relatable pain points** (3am, all-nighters, no concept)

### What's Weak
- **Generic formats** (comparison charts, arrow lists without visual punch)
- **Soft messaging** ("you're not bad at...", "push it further")
- **Text-heavy without visual hierarchy** (needs more scale contrast)
- **Gimmicks** (emoji diagrams feel forced)

### Recommendations for Testing
1. **Lead with Tier 1** - Test 04, 10, 03, 08 first
2. **A/B test tone** - Harsh truth (03, 04) vs. honest/helpful (08)
3. **Save Tier 3 for retargeting** - Softer messages work better warm
4. **Kill the roast** - Ad 13 is the most likely to bomb
